HTML은 Hyper Text Markup Language 의 약자이다.

하이퍼 텍스트 는 현재페이지에서 다른 페이지로 이동할 수 있는 링크를 말한다.
마이업은 웹 페이지의 구조를 설계할 떄 사용되는 언어라고 할 수 있다 . 

즉 , Html 은 웹페이지 구조 설계를 위한 페이지 이동 간으한 언어이다..

이때 html 을 구성하는 요쇼들을 태그(tag)라고 하는데 , 테그는 페이지를 구성하는 기능으로 HTML 의 핵심
<태그명> 내용 </태그명>

Html 과 태그명

<html>
    <head>
        (문서의 기본 정보를 작성한다.)
    </head>
    <body>
        (문서화면에서 보이는 모든 내용들을 작성한다.)
    </body>
</html>

<html> ~ </html> hmtl 의 문서의 시작과 끝을 알리는 태그.
<head> ~ </head> 직역하면 머리를 나타내며 , html 문서의 정보를 담고있는 부분이다.
<body> ~ </body> 직역하면 몸통이다 . 사람ㄴ의 몸을 생각한다면 모든 보이는 행동들을 다 처리한다고 보면 된다
                 HTML 문서로보면 실제문서상에 모든 기능을 처리.

CSS란 Cascading Style Sheets 의 약자로 그냥 스타일 시트라고 부른다 스타일을 설정한다는 것은 HTML 로 만든 페이지의 레이아웃과 모양을 훨씬 더 좋게 만드는 기술
웹페이지 작성시 반복적으로 사용되는 서식을 미리 설정하거나 글자, 색상, 프레임등을 꾸밀 수 있다. 

CSS에서 스타일의 내용을 만든후 이것을 어떤 요소에 어떻게 적용할 것인가가 중요.
태그선택자,아이디선택자,클래스 선택자가 있다.

</style>
